# Studien-Dashboard

Kleines Python-Programm für die Portfolio-Abgabe im Modul DLBDSOOFPP01_D.
Das Dashboard verwaltet einfache Studiendaten, Module, Noten und den aktuellen
ECTS-Fortschritt.

GitHub: https://github.com/mavlonfox/Fox-Marvin_IU14103178_Python

## Start

```bash
python src/main.py
```

## Tests

```bash
python run_tests.py
```

## Hinweise

- Es werden nur Python-Standardbibliotheken verwendet.
- Die Daten liegen in `data/studien_dashboard.json`.
- Zum Speichern und Laden gibt es Buttons direkt in der Oberfläche.
- Das Programm braucht keine Admin-Rechte, keine Internetverbindung,
  keine Datenbank, kein Docker und keine Paketinstallation mit pip.
